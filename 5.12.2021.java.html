Java는 크게 다음과 같은 4가지 에디션으로 나뉜다. Java SE(Java Standard Edition / J2SE) 대부분의 사람들이 가장 많이 접하는 표준 에디션. Java의 핵심 API와 기능들을 제공한다. JDK 항목도 참고. Jakarta EE, 구 Java EE(Java Enterprise Edition / J2EE) 기업에서 운영하는 서버 페이지에 특화된 에디션이다. JSP와 서블릿을 비롯한 웹 애플리케이션 서버에 관련된 기술들이 포함되어 있다. Java ME(Java Micro Edition / J2ME) PDA나 셋톱박스, 센서 등의 임베디드 시스템 환경에 특화된 에디션이다. JavaFX 데스크톱 애플리케이션 개발 및 배포를 위한 에디션으로, 크로스플랫폼 이식과 GUI 라이브러리를 제공한다.
